# Compact study evidence

These files are the final aggregate summaries used by the canonical research post. They preserve
claim-bearing counts, gates, hashes, and verdicts without publishing model completions, prompts,
step logs, transcripts, or per-attempt run directories.

They are immutable historical evidence, not inputs to the reusable method. The runnable batteries,
simulator interfaces, verifier contracts, and publication logic remain under `src/proprio`; new
experimental records should be generated under `runs/`.
