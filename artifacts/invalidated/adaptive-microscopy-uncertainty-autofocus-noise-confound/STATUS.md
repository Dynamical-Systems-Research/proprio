# Invalidated autofocus-noise confound

This 60-case run isolated the specimen-dependent absolute-focus problem, but the noise
intervention was still applied before autofocus. Two intended five-repeat repairs were rejected
because the autofocus curve itself lost prominence; additional post-focus measurements could not
causally repair that upstream failure. The run is retained and excluded. The replacement
condition fixes autofocus at the calibrated noise setting and changes only repeated-measurement
noise.
