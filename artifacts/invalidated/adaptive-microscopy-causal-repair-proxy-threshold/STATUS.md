# Invalidated proxy-threshold causal study

The truthful arm cited all three temporal-uncertainty failures but invented a branch threshold on
the candidate-visible `relative_spread`, which is not calibrated to the verifier statistic. Its
3+5-frame repair remained ambiguous and correctly terminated HOLD. The binary arm happened to
pass, so the primary truthful-over-none causal criterion could not pass. The partial control matrix
is retained and excluded. The method now forbids proxy-threshold inference and directs the agent
to the source-bounded maximum evidence budget when no calibrated mapping exists.
