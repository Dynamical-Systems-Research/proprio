# Diagnostic judge calibration

These partial cases were used to make the semantic-review contract explicit. They are not a
reported validation result and have no aggregate summary. The final prompt was frozen only
after two rules were made unambiguous: fabricated provenance is independently disqualifying,
and an unavailable target execution (`HOLD`) can never be upgraded to `ACCEPT` by nominal
replay. Final judge metrology uses the separate confirmatory instrument panel.
